# Adviento Casa de mi Señor

Este repositorio contiene el código fuente para el sitio web del Adviento de Casa de mi Señor. Cada día durante el mes de diciembre, se publica un video especial con un mensaje inspirador y un pasaje bíblico relacionado.

## Estructura del Proyecto

- `app/pages/2025/`: Contiene las páginas del sitio web para el año 2025.
- `server/api/2025/videos.ts`: Archivo que define los videos del Adviento, incluyendo títulos, presentadores, pasajes bíblicos y URLs de los videos.

## Cómo Contribuir

1. Clona el repositorio.
2. Usa `bun` para instalar las dependencias y ejecutar el proyecto localmente.
3. Realiza tus cambios en una nueva rama.
4. Envía un pull request para revisión.

## Despliegue

El sitio web se despliega automáticamente a través de un servicio de integración continua cada vez que se realiza un push a la rama principal.
